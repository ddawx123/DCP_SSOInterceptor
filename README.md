# DCP_SSOInterceptor Introduction
DCP_SSOInterceptor（小丁工作室云平台之SSO拦截器） 是本人自编自导的小型SSO客户端（在领域里也可以叫做SSO拦截器），用于安装在需要整合登录的站点。
此处仅仅提供一些常见程序的Demo，其他小众的或者自己开发的程序请基于这些Demo自行二次开发！
